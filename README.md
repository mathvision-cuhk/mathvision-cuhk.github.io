# Install
```bash
sudo apt update
sudo apt upgrade
ruby --version
gem install jekyll bundler
bundle init
bundle install
```

# Run locally
```bash
/bin/bash --login
rvm use ruby-3.1.0
bundle exec jekyll serve
```
